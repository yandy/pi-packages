# pi-dingtalk

钉钉 workspace CLI skills 集合，供 AI 辅助钉钉应用开发使用。

涵盖钉钉 workspace 各类产品：多维表格、即时通讯、日历、文档、云盘、待办、通讯录、考勤等。

## 前置条件

全局安装 `dingtalk-workspace-cli`：

```bash
npm install -g dingtalk-workspace-cli
```

## 安装

```bash
pi install npm:@yandy0725/pi-dingtalk
```

## 开发

```bash
# 从本地 DingTalk CLI 安装复制 skills
npm run download-skills
```
