# pi-lark

飞书（Lark）开放平台 API skills 集合，供 AI 辅助飞书应用开发使用。

涵盖飞书开放平台各类 API：多维表格、文档、电子表格、日历、即时通讯、审批、云盘、知识库等。

## 前置条件

全局安装 `@larksuite/cli`：

```bash
npm install -g @larksuite/cli
```

## 安装

```bash
pi install npm:@yandy0725/pi-lark
```

## 开发

```bash
# 从飞书开放平台下载/更新 skills
npm run download-skills
```
